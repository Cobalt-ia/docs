Cobalt vous aide à **cartographier un compte client** et à **identifier les contacts manquants** pour développer le compte (upsell).

## La démarche

- **Cartographier le compte** — visualisez les équipes, décideurs et contacts déjà connus de l'entreprise.
- **Identifier les contacts manquants** — repérez les décisionnaires que vous n'avez pas encore approchés.
- **Déclencher l'upsell** — engagez ces nouveaux contacts via vos workflows d'approche.

Voir [People Search Engine](/fr/fonctionnalites/people-search) pour trouver les bons décisionnaires, et [Développer vos comptes clients](/fr/cas-usage/comptes-clients) pour le playbook complet.
