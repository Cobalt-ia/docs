Les **templates de messages** sont des modèles réutilisables que vous insérez dans vos workflows, notamment dans les blocs de communication.

Ils existent par **type de canal** :

- **Email**
- **LinkedIn**
- **WhatsApp**

<img src="/images/template-messages.png" alt="Éditeur de modèle de message avec les variables disponibles" />

Chaque template peut contenir **toutes les variables disponibles dans Cobalt** (`{firstName}`, `{title}`, `{availability}`…), remplacées par les données réelles au moment de l'envoi. Vous pouvez aussi insérer des variables dans l'**objet** d'un template d'email.

<Note>
  La possibilité d'**ajouter des fichiers** aux templates arrive bientôt.
</Note>
