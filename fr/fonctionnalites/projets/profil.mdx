Un projet **Sourcer sur profil** vous permet de **sourcer en continu** sur un **type de profil** ou un **secteur**, sans être rattaché à un besoin client précis.

Idéal pour constituer un **vivier** sur les profils que vous placez régulièrement : à la création, vous décrivez le **profil recherché** (poste, compétences, séniorité, zone). Cobalt alimente le projet avec les profils correspondants via le [People Search Engine](/fr/fonctionnalites/people-search/people-search).

Comme pour le sourcing sur besoin, la vue **Kanban** est la vue par défaut.
