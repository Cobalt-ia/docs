Un projet **Sourcer sur besoin** vous permet de **sourcer des candidats** pour répondre à un **besoin client précis**.

À la création, vous rattachez le **besoin** à pourvoir (intitulé, compétences, critères). Cobalt s'appuie dessus pour vous proposer les profils les plus pertinents via le [People Search Engine](/fr/fonctionnalites/people-search/people-search).

La vue **Kanban** (colonnes par statut) est la vue par défaut : faites avancer chaque candidat dans votre pipeline de sourcing.
