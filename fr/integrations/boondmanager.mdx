Cobalt dispose d'une intégration native avec BoondManager. Une fois connectés, candidats, clients, besoins et entreprises circulent automatiquement entre les deux plateformes en temps réel.

## Ce qui se synchronise

La synchronisation est **bidirectionnelle** et en **temps réel** pour :

- Candidats
- Clients
- Besoins
- Entreprises

<Info>
  Toute modification effectuée d'un côté apparaît de l'autre en quelques secondes. Aucune synchronisation planifiée n'est nécessaire.
</Info>

## Comment connecter

<Tip>
  Connectez BoondManager avant d'importer d'autres données afin d'éviter les doublons.
</Tip>

<Steps>
  <Step title="Ouvrir les intégrations">
    Dans Cobalt, accédez à **Paramètres → Intégrations** et cliquez sur **BoondManager**.
  </Step>
  <Step title="Renseigner les 3 identifiants">
    La connexion requiert trois identifiants :

    - **Jeton client**
    - **Clé client**
    - **Jeton utilisateur**

    Le **jeton client** et la **clé client** se trouvent dans BoondManager sous **Administration → Espace développeur → API / Sandbox**.

    Le **jeton utilisateur** se trouve dans **Mon compte**, en ouvrant le menu déroulant **API**.

    <img src="/images/boondmanager-identifiants.png" alt="Modale d'intégration BoondManager : jeton client, clé client, jeton utilisateur et Light mode" width="420" />
  </Step>
  <Step title="(Optionnel) Activer le Light mode">
    Le **Light mode** limite la consommation d'API lors de la synchronisation : il ignore les données les moins importantes (par exemple les avatars). Recommandé si votre forfait BoondManager comprend peu d'appels API et une grande base de données. Vous retrouvez votre consommation dans l'**espace développeur** de BoondManager.
  </Step>
  <Step title="Lancer la synchronisation">
    Une fois les trois jetons renseignés, lancez la synchronisation. Selon la taille de votre base, elle prend entre **2 minutes** (moins de 5 000 enregistrements) et **1 heure** (plus de 100 000). Tout se fait de manière **asynchrone** : vous n'avez rien à faire de votre côté.
  </Step>
</Steps>
