## Questions fréquentes

Zoho Recruit est-il moins cher que Cobalt ?

Oui, Zoho Recruit est moins cher en entrée de gamme : $25/user/mois (Standard) vs 31€/user/mois (Cobalt Starter). Mais Zoho Recruit offre moins de fonctionnalités IA avancées. Cobalt inclut natively agent IA Balt autonome, transcription meetings, dossiers compétences automatisés — ce qui justifie l'écart pour ESN et cabinets.

Qu'est-ce que Zoho Zia et comment se compare-t-il à Cobalt AI ?

Zoho Zia est l'IA générale de toute la suite Zoho (CRM, Recruit, Desk, Books, etc.) : suggestions, automation basique, scoring, anomaly detection. Elle n'est pas spécialisée recrutement. Cobalt AI-first est conçu spécifiquement pour le recrutement avec recherche sémantique native, agent IA autonome Balt, dossiers automatisés, transcription. L'IA Cobalt est plus aboutie pour les cas d'usage recrutement.

Zoho Recruit s'intègre-t-il à Boondmanager ?

Non, Zoho Recruit n'a pas d'intégration native avec Boondmanager. Zoho Recruit est très intégré avec la suite Zoho One (CRM, Books, Projects) mais pas avec l'écosystème staffing français. Cobalt est le seul ATS en 2026 avec intégration Boondmanager native.

Zoho Recruit est-il hébergé en Europe ?

Zoho propose des data centers en Europe (Pays-Bas) avec option de résidence EU pour la conformité RGPD, mais il faut le configurer explicitement. L'hébergement par défaut peut être USA/Inde selon votre géographie. Cobalt est nativement hébergé en Europe (AWS/Scaleway EU) sans configuration supplémentaire.

Pour qui Zoho Recruit est-il le meilleur choix ?

Zoho Recruit est pertinent pour les entreprises qui utilisent déjà la suite Zoho One (CRM, Books, Projects, etc.) et cherchent un ATS économique pour du recrutement interne standard. L'intégration native Zoho One est un argument fort. Pour ESN, cabinets de recrutement et organisations cherchant IA end-to-end avec intégration européenne, Cobalt est plus adapté.

L'UX de Zoho Recruit est-elle moderne ?

Zoho Recruit a une interface héritée de 2018 avec des rafraîchissements UI réguliers mais une architecture datée. Les utilisateurs rapportent une courbe d'apprentissage importante et des écrans denses. Cobalt propose une UX 2025 repensée : épurée, rapide (< 500ms par action), intuitive avec onboarding guidé < 1h.

Comment migrer de Zoho Recruit vers Cobalt ?

Cobalt propose une migration gratuite et assistée depuis Zoho Recruit en 3-6 semaines : export via API Zoho, nettoyage IA automatique, mapping des champs et pipelines, import candidats/jobs/évaluations, formation équipe (2h), parallel run 2-3 semaines, puis bascule. Toutes les données sont préservées.
