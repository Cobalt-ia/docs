Cobalt supports **versioning** for skills documents: you can create **several documents for the same candidate**.

This is useful, for example, to tailor a candidate's presentation to the client or need at hand, or to keep an enriched version after an interview.

## Link a document to a need

A skills document can be **linked to a need**. This keeps the connection between the document version and the client need it answers.
